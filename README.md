# React Analytics Dashboard

A simple analytics dashboard built using React.js to visualize key product and business metrics.

## Features
- Responsive analytics dashboard UI
- Reusable React components for metrics and charts
- Data visualization using Recharts
- Clean component-based architecture

## Tech Stack
- React.js
- JavaScript (ES6+)
- Recharts
- CSS
- Vite

## Screens
- Metrics overview (Users, Revenue, Orders, Conversion Rate)
- Line chart showing trends over time

## How to Run Locally
```bash
npm install
npm run dev
